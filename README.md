# My Transformation

> **Note:** This repository contains the **Portfolio Edition** of the application, prepared for demonstration and recruitment purposes. It showcases the complete transformation tracking workflow while using fictional data to protect personal information.

*A modern transformation journal that helps users track weight loss, body measurements, milestones and long-term progress in one structured workspace.*

# ✨ Features

- Transformation Dashboard
- Weight & body measurement tracking
- Progress photo gallery
- Personal journal & milestones
- Interactive charts & statistics
- Goal progress tracking
- Transformation calendar
- AI-assisted Quick Import
- Local-first architecture
